Privacy Policy

Effective date: March 13, 2026

This Privacy Policy explains how this internal automation tool handles information. This tool is private and is not available to the public.

1. What this tool is

This is a private n8n-based automation workflow used exclusively by its owner to publish AI-generated video content to their own social media accounts (TikTok, YouTube, and Instagram). It is not a product, SaaS platform, or public service.

2. Data collected

This tool does not collect, store, or process personal data belonging to third parties. The only data processed is:

  • Content created by the owner (video scripts, topics, metadata)
  • API tokens and credentials belonging to the owner's own platform accounts
  • Workflow execution logs generated by n8n for operational purposes

3. Data from connected platforms

When interacting with TikTok, YouTube, or Instagram APIs, the tool may receive API responses containing account information. This data is:

  • Used only to confirm successful content publication
  • Not stored beyond the current workflow execution
  • Not shared with any third parties

4. No third-party data sharing

This tool does not sell, rent, trade, or otherwise share any data with third parties. No advertising, analytics, or tracking services are used.

5. Data storage

Workflow inputs and outputs may be temporarily stored within the n8n instance. No personal data of third parties is stored. API credentials are stored securely as environment variables and are not logged or exposed.

6. Security

API keys and access tokens are stored as environment variables within the private n8n instance. Access to the tool is restricted to the owner only.

7. Your rights

As this tool processes only the owner's own data, no third-party rights or data subject requests apply. If you believe this tool has processed your data in error, please contact the owner directly.

8. Changes to this policy

This policy may be updated at any time. The effective date at the top of this document will reflect the most recent revision.

9. Contact

For any questions about this privacy policy, please contact the tool owner directly.
